Bio Ton Tile Server

This page demonstrates hosting tiles for OpenStreetMap/umap integration.

Tile URL Pattern:

https://fkistner85.github.io/Bio_Ton/tiles/{z}/{x}/{y}.png

Usage in umap:

1. Open umap editor
2. Click on "Change tile layers"
3. Add custom layer with the URL above
4. Adjust attribution if needed
