Replace {z}, {x}, and {y} with zoom level, x-coordinate, and y-coordinate respectively.

Integration with umap

umap is a tool to create maps with OpenStreetMap layers and embed them in your site.

Step 1: Create or Open a Map

  • Go to umap.openstreetmap.fr
  • Sign in with your OpenStreetMap account
  • Click "Create a map" or open an existing map

Step 2: Add Custom Tile Layer

  • Click the layers icon on the right side
  • Click on "Add a remote layer" or "Change tile layers"
  • Select "Custom" from the dropdown
  • Enter the tile URL: https://fkistner85.github.io/Bio_Ton/tiles/{z}/{x}/{y}.png
  • Set the name (e.g., "Bio Ton Tiles")
  • Set attribution if required
  • Click "Save"

Step 3: Adjust Layer Settings

  • Set opacity if you want the layer to be transparent
  • Choose whether it should be a base layer or overlay
  • Save your map
üí° Tip: You can add multiple tile layers and switch between them in the umap interface.

Integration with Leaflet.js

Leaflet is a popular open-source JavaScript library for interactive maps.

Basic Setup

<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="https://unpkg.com/leaflet@1.9.4/dist/leaflet.css" />
    <script src="https://unpkg.com/leaflet@1.9.4/dist/leaflet.js"></script>
    <style>
        #map { height: 600px; }
    </style>
</head>
<body>
    <div id="map"></div>
    <script>
        var map = L.map('map').setView([51.505, -0.09], 13);
        
        // Add Bio Ton tiles
        L.tileLayer('https://fkistner85.github.io/Bio_Ton/tiles/{z}/{x}/{y}.png', {
            attribution: 'Bio Ton Tiles',
            maxZoom: 18,
            minZoom: 0
        }).addTo(map);
    </script>
</body>
</html>

As an Overlay Layer

// Base layer (OpenStreetMap)
var baseLayer = L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
    attribution: '¬© OpenStreetMap contributors'
}).addTo(map);

// Overlay layer (Bio Ton tiles with transparency)
var overlayLayer = L.tileLayer('https://fkistner85.github.io/Bio_Ton/tiles/{z}/{x}/{y}.png', {
    attribution: 'Bio Ton Tiles',
    opacity: 0.7
}).addTo(map);

// Add layer control
L.control.layers(
    { "OpenStreetMap": baseLayer },
    { "Bio Ton Overlay": overlayLayer }
).addTo(map);

Integration with OpenLayers

OpenLayers is a high-performance, feature-packed library for creating interactive maps.

Basic Setup

<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/ol@v7.3.0/ol.css">
    <script src="https://cdn.jsdelivr.net/npm/ol@v7.3.0/dist/ol.js"></script>
    <style>
        #map { width: 100%; height: 600px; }
    </style>
</head>
<body>
    <div id="map"></div>
    <script>
        var map = new ol.Map({
            target: 'map',
            layers: [
                new ol.layer.Tile({
                    source: new ol.source.XYZ({
                        url: 'https://fkistner85.github.io/Bio_Ton/tiles/{z}/{x}/{y}.png',
                        maxZoom: 18
                    })
                })
            ],
            view: new ol.View({
                center: ol.proj.fromLonLat([-0.09, 51.505]),
                zoom: 13
            })
        });
    </script>
</body>
</html>

QGIS Integration

You can also use these tiles in QGIS desktop application:

  1. Open QGIS
  2. Go to Layer ‚Üí Add Layer ‚Üí Add XYZ Layer
  3. Create a new connection with these settings:
    • Name: Bio Ton Tiles
    • URL: https://fkistner85.github.io/Bio_Ton/tiles/{z}/{x}/{y}.png
    • Min Zoom: 0
    • Max Zoom: 18
  4. Click OK and add the layer to your map

Troubleshooting

‚ö†Ô∏è Common Issues:
  • Tiles not loading: Ensure tiles are properly extracted and committed to the repository
  • 404 Errors: Verify the tile path structure matches {z}/{x}/{y}.png
  • CORS Errors: GitHub Pages should handle CORS automatically. Check browser console for details.
  • Blank tiles: Make sure your zoom level is within the range of available tiles
